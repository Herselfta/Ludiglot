# Font Library Directory

This directory is used to store font files for the Ludiglot application.

## Usage

Place your preferred font files (`.ttf`, `.otf`) here and they will be automatically detected by the application.

The font selection menu in the application will list all available fonts in this directory.

## Supported Formats

- TrueType Font (`.ttf`)
- OpenType Font (`.otf`)

## Notes

- Font files are **not** tracked by git (see `.gitignore`)
- Default font: Source Han Serif SC (思源宋体)
